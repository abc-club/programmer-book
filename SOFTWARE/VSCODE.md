* ctrl+p

#### prettier

https://prettier.io/

1.  创建配置文件(优先级从上到下)
    .prettierrc
    .editconfig
    编辑器默认配置

```
// .prettierrc
{
  "tabWidth": 2,
  "singleQuote": true,
  "trailingComma": "all",
  "semi": false
}
```

**语法可参考官网或者插件说明**

2.  修改编辑器用户设置，把 formatOnSave 设置为 true，表示在保存时格式化代码
    "editor.formatOnSave": true
